## ?? Quick Reference

| Aktion | Tastenkombination | Menü |
|--------|-------------------|------|
| **Starten** | `F5` | Debug ? Start Debugging |
| **Stoppen** | `Shift + F5` | Debug ? Stop Debugging |
| **Neu builden** | `Ctrl + Shift + B` | Build ? Rebuild Solution |
| **Solution Explorer** | `Ctrl + Alt + L` | View ? Solution Explorer |
